Project started on 03/15/2021 - ended on 03/20/2021 
Used langages : HTML, PHP, SQL

Mini chat features :
- MVC architecture
- Remember pseudo using cookie
- Date and time of sent messages
- Display old messages on different pages (automatic pagination)
- Code secure using htmlspecialchars, prepared statement and isset/empty/interval condition on POST and GET variables
- Notify and doesn't add message if pseudo or message are empty

Used tools :
- PhpStorm
- WAMP
